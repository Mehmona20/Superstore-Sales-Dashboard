# Superstore-Sales-Dashboard
Power BI dashboard for sales and profit analysis
This project is a Power BI dashboard created using the Superstore dataset, which was obtained from Kaggle. The data was first imported into SQL and then loaded into Power BI for analysis. The dashboard presents key business insights including total sales, profit, and quantity through KPI cards. It also shows sales and profit trends over time, category-wise comparison, and segment-wise distribution. The purpose of this dashboard is to analyze business performance and identify important patterns and trends in the data.
